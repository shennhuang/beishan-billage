# Beishan Village — 購物車靜態網頁

精選 3C 好物購物車網頁，部署於 GitHub Pages。

## 功能

- 🛍️ 商品列表展示（標題、圖片、金額、庫存）
- 🛒 購物車側欄（加入/刪除/調整數量）
- 💰 自動計算總金額
- 📋 結帳訂單確認 Modal
- 💾 localStorage 持久化
- 📱 完整響應式設計

## 技術

- HTML5 + Vanilla CSS + Vanilla JavaScript
- 無框架依賴，純靜態部署

## 部署

1. 將此資料夾推送至 GitHub Repository
2. 進入 Settings → Pages → 選擇 `main` 分支
3. 網站即可透過 `https://<username>.github.io/beishan-farm/` 訪問
